# Python Learning Journey

Projects built while learning Python from scratch.

## Projects

### Calculator
A CLI calculator with addition, subtraction, multiplication and division.
Features error handling, confirmation prompts, and persistent history.

### Weather App
Live weather fetcher using Open-Meteo API.
Enter any city and get real-time temperature and wind data.

### Student Report Card
OOP project that calculates total, average, highest and lowest marks.
Generates a formatted report with grade.

## Skills Used
Python, OOP, APIs, File I/O, Error Handling, Git
